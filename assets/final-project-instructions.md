# 🏆 Final Mini-Project Instructions

## Your Mission: Build a Personal Portfolio Page!

You've learned HTML, CSS, JavaScript, responsive design, and API integration. Now put it all together!

---

## 📋 Requirements

Your portfolio page must include:

### Must-Have (Required)
- [ ] A **header** with your name and a short tagline
- [ ] An **About Me** section with at least 2 paragraphs
- [ ] A **Skills** section showing what you learned
- [ ] At least **one interactive element** (counter, color changer, etc.)
- [ ] **Responsive design** — looks good on mobile AND desktop
- [ ] Proper use of **semantic HTML** (header, main, section, footer)
- [ ] An external **CSS file** (not inline styles)
- [ ] An external **JavaScript file**

### Nice-to-Have (Bonus Points!)
- [ ] Dark mode toggle
- [ ] An API-powered feature (jokes, quotes, dog photos, etc.)
- [ ] Smooth scroll navigation
- [ ] Hover animations on cards or buttons
- [ ] A contact section
- [ ] Custom color scheme
- [ ] At least 3 different font sizes
- [ ] A "Projects" section showcasing what you built in earlier modules

---

## 🎯 Step-by-Step Guide

### Step 1: Plan (5 minutes)
- Sketch your layout on paper
- Decide what sections you want
- Pick a color scheme (try [coolors.co](https://coolors.co) for inspiration)

### Step 2: HTML Structure (15 minutes)
- Create `index.html`
- Add all your sections with proper tags
- Don't worry about styling yet — just get the content in!

### Step 3: CSS Styling (20 minutes)
- Create `style.css` and link it
- Start with mobile styles (mobile-first!)
- Add colors, fonts, spacing
- Add media queries for larger screens

### Step 4: JavaScript (15 minutes)
- Create `script.js` and link it at the bottom of your HTML
- Add at least one interactive feature
- Test it works!

### Step 5: Polish (10 minutes)
- Check it on different screen sizes (resize your browser)
- Fix any spacing issues
- Make sure all buttons/links work
- Add final touches

## 💡 Ideas for Interactive Features

Pick one (or more!) of these:

| Feature | Difficulty | What it does |
|---------|-----------|--------------|
| Click counter | ⭐ Easy | Counts button clicks |
| Color changer | ⭐ Easy | Random background colors |
| Dark mode toggle | ⭐⭐ Medium | Switches light/dark theme |
| Typewriter effect | ⭐⭐ Medium | Text appears letter by letter |
| Joke fetcher | ⭐⭐ Medium | Gets jokes from an API |
| Quote of the day | ⭐⭐ Medium | Displays random quotes |
| To-do list | ⭐⭐⭐ Hard | Add/remove items from a list |
| Mini quiz | ⭐⭐⭐ Hard | Questions with score tracking |

---

## 🎨 Design Tips

- **Keep it simple** — White space is your friend!
- **Max 2-3 colors** — Pick a primary color and use shades of it
- **Max 2 fonts** — One for headings, one for body text
- **Consistency** — Same spacing, same border-radius everywhere
- **Mobile first** — Design for phones, then add for bigger screens
