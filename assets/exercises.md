# 🏋️ Quick Exercises (10 per Module)

Complete these exercises to practice what you've learned!
Start easy, get harder. No peeking at answers until you've tried!

---

## Module 1: HTML Basics

1. **My First Page** — Create a page with your name as the heading and a paragraph about yourself.
2. **Favorite List** — Make an unordered list of your 5 favorite movies/songs/games.
3. **Recipe Card** — Write a recipe using headings, a paragraph, and an ordered list of steps.
4. **Link Collection** — Create 5 links to your favorite websites, each on a new line.
5. **Image Gallery** — Add 3 images to a page (use placeholder.com or any image URL).
6. **All the Headings** — Use all 6 heading sizes (h1 to h6) to show a hierarchy (like a book's table of contents).
7. **About My School** — Create a page about your school with sections for: Name, Location, What I like about it.
8. **Two Lists** — Make one ordered list (top 3 goals) and one unordered list (hobbies).
9. **Bold & Italic** — Write a paragraph that uses both `<strong>` (bold) and `<em>` (italic).
10. **Complete Template** — From memory, write a complete HTML page (doctype, html, head, title, body) without looking at examples.

---

## Module 2: CSS Styling

1. **Color Party** — Change the background color of the page and make all text a different color.
2. **Card Design** — Create a white box with rounded corners, a shadow, and padding.
3. **Hover Magic** — Make a button change color when you hover over it.
4. **Font Explorer** — Try 3 different font-families and pick your favorite.
5. **Border Bonanza** — Create 3 boxes: one with a solid border, one dashed, one dotted.
6. **Spacing Master** — Create two boxes and control the space between them (margin) and inside them (padding).
7. **Class vs ID** — Style one element with a class and another with an ID. See the difference!
8. **Rainbow Text** — Make each line of a paragraph a different color using `<span>` with classes.
9. **Button Collection** — Design 3 different button styles (primary, secondary, danger).
10. **Dark Theme** — Restyle your Module 1 page with a dark background and light text.

---

## Module 3: JavaScript Interaction

1. **Alert Button** — Make a button that shows an alert saying "Hello!" when clicked.
2. **Name Display** — Get text from an input and display it on the page when a button is clicked.
3. **Background Changer** — Make a button that changes the page's background to a random color.
4. **Character Counter** — Show how many characters are typed in a text input (live update).
5. **Toggle Text** — Make a button that shows/hides a paragraph.
6. **Simple Counter** — Create buttons to count UP and DOWN, display the number.
7. **Disable Button** — Make a button that disables itself after being clicked 5 times.
8. **Clock Display** — Show the current time on the page (bonus: update every second).
9. **List Adder** — Type text in an input, click a button, and it adds to a list on the page.
10. **Password Checker** — Check if a typed password is longer than 8 characters, show "Strong" or "Weak".

---

## Module 4: Responsive Design

1. **Flex Row** — Create 3 boxes in a row using flexbox.
2. **Centered Box** — Center a box both horizontally and vertically using flexbox.
3. **Card Wrap** — Make 6 cards that show 3 per row on desktop and 1 per row on mobile.
4. **Responsive Nav** — Create a navigation that's horizontal on desktop, vertical on mobile.
5. **Image Fit** — Make an image that's never wider than its container.
6. **Two Columns** — Create a 2-column layout that stacks on mobile.
7. **Font Sizing** — Make headings smaller on mobile using media queries.
8. **Hide on Mobile** — Have an element that only appears on screens wider than 768px.
9. **Spacing Adjust** — Use different padding values for mobile vs desktop.
10. **Full Layout** — Create a header, 3-column content area, and footer that collapses responsively.

---

## Module 5: API Integration

1. **Fetch & Log** — Fetch data from any API and console.log the result.
2. **Display Data** — Fetch a joke and display it in a paragraph element.
3. **Loading State** — Show "Loading..." text while waiting for API data.
4. **Error Handling** — Add try/catch and show a friendly error message if the fetch fails.
5. **Button Trigger** — Only fetch data when a button is clicked (not on page load).
6. **Multiple Fetches** — Fetch from two different APIs and display both results.
7. **Image Fetch** — Fetch a random image URL from an API and display the image.
8. **Refresh Button** — Add a "Get Another" button that fetches new data each time.
9. **Disable While Loading** — Disable the fetch button while data is loading.
10. **User Input Fetch** — Let the user type a number, then fetch a fact about that number.

## Module 6: Mini Project

1. **Add Your Name** — Customize the portfolio header with your real name.
2. **New Section** — Add a "Hobbies" or "Favorites" section with at least 3 items.
3. **Color Scheme** — Change the color palette to your favorite colors.
4. **New Skill** — Add a new skill card to the skills grid.
5. **Project Card** — Add a real project you made during this workshop.
6. **Animation** — Add a CSS hover animation to one of the cards.
7. **New API Feature** — Add a new API-powered feature (quotes? cat facts? trivia?).
8. **Custom Font** — Add a Google Font to make the page look unique.
9. **Contact Form** — Replace the contact links with a simple form (name + message).
10. **Deploy It!** — Ask your facilitator about putting it online with GitHub Pages!

## 💡 Tips for Exercises

- Start with what you know, then look up what you don't
- It's OK to look at previous module code for reference
- Use the browser console (F12) to test ideas
- If stuck for more than 5 minutes, ask for a hint!
- There's no single "right answer" — if it works, it works!
