Welcome to the **Frontend Development Workshop**! 🎉

## ⚡ Quick Start (Automated Setup)

**Windows users:** Just double-click `setup.bat` and it will:
- ✅ Check if Node.js, npm, and VS Code are installed
- ✅ Install recommended VS Code extensions (Live Server, Prettier, HTML CSS Support)
- ✅ Install React demo dependencies
- ✅ Print a summary and next steps

After running setup, you're ready to go!

## 🚀 How to Run Each Section

### Option 1: Automated Setup (Easiest!)

1. Double-click `setup.bat` in the project root
2. Follow the on-screen instructions
3. Open any `index.html` with Live Server and start learning!

### Option 2: VS Code + Live Server (Manual)

1. Download and install [VS Code](https://code.visualstudio.com/)
2. Open this folder in VS Code
3. Install the **Live Server** extension (click Extensions icon → search "Live Server" → Install)
4. Right-click any `index.html` file → "Open with Live Server"
5. Your browser will open with the page! 🎊

### Option 3: Just Double-Click

You can also just double-click any `index.html` file and it will open in your browser. Simple!

### Option 4: React Demo (requires Node.js)

Only needed for the optional `react-demo/` folder:

1. Install [Node.js](https://nodejs.org/) (download the LTS version)
2. Open a terminal in the `react-demo/` folder
3. Run: `npm install`
4. Run: `npm run dev`
5. Open the link shown in the terminal

## 🆘 Troubleshooting

| Problem | Solution |
|---------|----------|
| Page is blank | Check for typos in your HTML tags |
| CSS not working | Make sure your `<link>` tag path is correct |
| JavaScript not running | Open browser console (F12) and check for red errors |
| Live Server not starting | Make sure you right-clicked an HTML file |
| React won't start | Make sure you ran `npm install` first |
| Nothing shows up | Try refreshing the page (Ctrl+R or Cmd+R) |
| setup.bat says Node missing | Download from [nodejs.org](https://nodejs.org/) and rerun |
| setup.bat says VS Code missing | Download from [code.visualstudio.com](https://code.visualstudio.com/) |

## 💡 Tips for Success

- **Don't be afraid to experiment!** You can't break anything permanently.
