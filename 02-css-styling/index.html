What is CSS?

CSS stands for Cascading Style Sheets. It controls colors, fonts, spacing, layout, and more!

How CSS Works

You write rules that say: "Hey browser, make all paragraphs blue" or "Make this box have rounded corners".

Three Ways to Add CSS

  • External file (like we're doing here!) — Best practice ✅
  • Internal (inside a <style> tag) — OK for small stuff
  • Inline (directly on an element) — Avoid if possible ❌

This box uses an ID selector. IDs are unique — only ONE element can have each ID!
