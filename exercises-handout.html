🏋️ Workshop Exercises Handout

Complete each exercise after the presenter finishes that module. Check them off as you go!

📄 Module 1: HTML Basics

⏱️ Time: 5 minutes | 📂 File: 01-html-basics/index.html

Tasks:

  1. Change the <h1> text to say your own name (e.g., "Welcome to Alex's Page!")
  2. Add an unordered list (<ul>) with your 3 favorite foods
  3. Add a link (<a>) to your favorite website
  4. Add a second paragraph about your hobbies using <p>
  5. Save (Ctrl+S) and check in your browser!
⭐ Bonus: Add an image using <img src="https://via.placeholder.com/200" alt="My image">
💡 Hint: Tags come in pairs! <p>Your text here</p>. Save often and refresh!

🎨 Module 2: CSS Styling

⏱️ Time: 7 minutes | 📂 File: 02-css-styling/style.css

Tasks:

  1. Change the background-color of body to a color you like (try #fff0f5 or lightyellow)
  2. Change the .card border color to your favorite color
  3. Make the h1 font-size bigger (try 3rem)
  4. Change the .cool-button background color
  5. Change what happens on .cool-button:hover (try a different color!)
⭐ Bonus: Add box-shadow: 0 4px 15px rgba(0,0,0,0.2); to .card for a fancy shadow effect!
💡 Hint: CSS format is property: value; — don't forget the semicolon! Try color names like "coral", "teal", "orchid".

⚡ Module 3: JavaScript

⏱️ Time: 8 minutes | 📂 File: 03-javascript-interaction/script.js

Tasks:

  1. Open the page in Live Server and test all the buttons
  2. In script.js, find the messages array and add your own messages
  3. Find the greeter function and add a "Good night" greeting for after 9pm (hour >= 21)
  4. Change the random color function to only use your favorite colors
  5. Open the browser Console (F12) and type alert("Hello!")
⭐ Bonus: Add a "Reset" button for the counter. You'll need:
• A new <button id="reset-btn">Reset</button> in the HTML
• JavaScript to set count = 0 and update the display when clicked
💡 Hint: The Console (F12) is your best debugging friend. Red text = error. Look at the line number!

📱 Module 4: Responsive Design

⏱️ Time: 7 minutes | 📂 File: 04-responsive-design/style.css

Tasks:

  1. Open the page and resize your browser window — watch cards rearrange!
  2. Press F12 → click the "Toggle Device Toolbar" icon (📱) to simulate a phone
  3. In style.css, find @media (min-width: 768px)
  4. Change 768px to 500px — resize and see the difference!
  5. Add a 5th card in the HTML — does the layout still look good?
⭐ Bonus: Add a media query that makes h1 font-size 1.2rem on mobile and 2.5rem on desktop.
💡 Hint: Mobile-first means: write small screen styles first, then use @media (min-width: ...) for bigger screens.

🌐 Module 5: API Integration

⏱️ Time: 8 minutes | 📂 File: 05-api-integration/script.js

Tasks:

  1. Open the page and click "Tell Me a Joke" — a joke appears from the internet!
  2. Click "Show Me a Dog" — a random photo loads!
  3. Open DevTools (F12) → Network tab → click a button → see the API call!
  4. In script.js, find fetchJoke and change the category (try Programming or Christmas)
  5. Find the dog fetch and try the URL in a new browser tab to see raw JSON
⭐ Bonus: Add a new button that fetches cat facts from https://catfact.ninja/fact
• The response has data.fact containing the text
• Copy the pattern from fetchJoke and adapt it!
💡 Hint: Always use try/catch with fetch()! The internet isn't always reliable.

🏗️ Module 6: Mini Project

⏱️ Time: 15 minutes | 📂 File: 06-mini-project/

Tasks:

  1. Open 06-mini-project/index.html in Live Server
  2. Find "Your Name" in the hero section and replace with your real name
  3. Update the "About Me" section with 2 sentences about yourself
  4. Add a new skill card (copy an existing one and change the emoji + text)
  5. Test the Dark Mode toggle — does it work?
  6. Click around the Fun Zone (jokes, counter, colors)
  7. Change at least 2 colors in style.css
⭐ Bonus Challenges:
• Add a new section (Hobbies? Favorite Music? Goals for this year?)
• Add another API feature to the Fun Zone
• Change the gradient colors in the hero section
• Add hover animations to the skill cards
💡 Hint: This is YOUR portfolio — be creative! There are no wrong answers. Try things, break things, fix things. That's how real developers work!
